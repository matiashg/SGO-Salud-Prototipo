# SGO-Salud

## Descripción

SGO-Salud es un prototipo desarrollado en Java como proyecto integrador para la materia Seminario de Práctica de Informática.

El sistema permite administrar pacientes y turnos médicos utilizando una base de datos MySQL mediante JDBC.

## Tecnologías utilizadas

- Java 21
- Apache NetBeans IDE
- MySQL
- JDBC

## Funcionalidades

- Registrar pacientes.
- Buscar pacientes por DNI.
- Listar pacientes.
- Actualizar obra social.
- Registrar turnos médicos.
- Listar turnos.
- Uso de arreglos.
- Uso de ArrayList.
- Exportación de pacientes a archivo TXT.

## Base de datos

Importar el archivo `sgo_salud.sql` antes de ejecutar el proyecto.
